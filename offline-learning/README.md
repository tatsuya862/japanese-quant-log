# Quant Log Drive Study

高速道路の渋滞・トンネル内を想定した、オフライン音声学習サービスのプロトタイプです。

## 設計メモ

- 既存サイト配下の静的PWAとして公開
- 初回アクセス時にUI、教材本文、背景、音声ファイルをService Workerでキャッシュ
- 学習モードは `音声` と `文章`
- 音声は1本10-20分、文章は1ポスト5-10分読了を想定
- 教材テーマは「失敗ログ x 金融リテラシー」

## 実運用で差し替えるもの

- `assets/audio/sample-local.wav` を各教材の音声ファイルに差し替え
- `app.js` の `episodes` 配列に本番タイトル、本文、音声パスを追加
- トップページから `offline-learning/index.html` に誘導
