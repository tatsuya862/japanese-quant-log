# FXトレーダー向け集中タイマー

- 25分の集中（集中モード）と5分の休憩（休憩モード）を自動で繰り返すPomodoro風タイマー。FXトレード前のルーチンや集中セッションに使いやすいUI。
- Webブラウザで開けばそのまま利用開始できます。
- スタート/停止とリセットで手動操作可能。集中サイクル数も表示して連続セッションを把握。

## 使い方
1. `index.html` をブラウザで開きます（ローカルファイルでOK）。
2. 「開始」で25分カウントダウン開始。集中が終わると自動的に5分休憩に切り替わります。
3. 「停止/再開」で途中から再開、「リセット」で最初の集中25分に戻すことができます。
4. サイクル数と次の切り替え先（集中/休憩）を目安にリズムを整えましょう。

## 追加のカスタマイズ案
- 取引時間帯に合わせて集中/休憩の長さを調整する設定。
- 音や通知で切り替えを知らせる。
- セッション履歴を簡単にログ保存してトレード全体の時間管理に活用。
