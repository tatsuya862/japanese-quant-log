# YouTubeライブ原稿（約10分）

## 0. 導入（〜1分）
- Quant Logの夜、いつも通り「量」と「質」を行き来する時間です。今日のテーマは失敗のログに刻まれていた“道具の欠如”をふまえた2つのアプリの話です。

## 1. 長時間集中のデメリット（〜2分）
- このチャンネルやデイリーログで何度も語ってきた通り、長時間チャートを見続けると意思決定の精度が落ちてしまう。休憩を入れないと雑な操作・細かい情報の見逃しが増え、翌日に疲れが残る悪循環に。
- 「今やらないと」のクセで集中→集中→疲弊になっていた自分の過去ログを引き合いにしながら、集中と休憩を意図的に切り替えないことの損失を共有する。

## 2. 家計管理をしていないことのデメリット（〜2分）
- 収入があっても使途を管理していないと、気づいたときには娯楽費が膨らんでたり、貯金が減っていたり。過去の配信ログで「なんとなく全額使っていた結果、予想外の支出に対応できなかった」エピソードを引く。
- 固定収入なのに自由度がないと感じるのは、数字を見ていないから。人生のイベントや投資チャンスに対して「残りの枠がどこにあるかわからない」という不安を共有。

## 3. アプリがないことで起きていたこと（〜1.5分）
- タイマーがないと集中のやりすぎ。家計アプリがないと使い道がバラバラ。どちらも現場の失敗（疲れ切って判断ミス、収支が曖昧で気づけば自由枠ナシ）につながっていた。
- 過去の配信で「気づけば数時間ぶっ通し」「収支を気にしていないから先延ばしになる」と報告した場面がまさにその証明。

## 4. 本日公開したタイマー＋家計アプリ紹介（〜3分）
- 【タイマー】　集中/休憩の自動切り替え、スタート/停止/リセットのみのライトUI。長時間のダラダラを防ぐために、直感的に操作できるインターフェースに。
- 【家計アプリ】　固定収入を入力するだけで、貯金20%、住宅25%、食費15%、娯楽10%に分けた予算が表示され、残り30%は自由枠。悩まずに内訳を可視化し、次の一手を考えられるように。
- 両者とも「ないと困っていた」状況の改善のためのツール。告知ではなく、失敗からの学びとして紹介。

## 5. クロージング（〜1.5分）
- ライブを振り返りつつ、今日の話を実生活に落とし込むヒント。「タイマーで質のリズムを」「家計アプリで金の流れを」整えて、次回またお会いしましょう。
